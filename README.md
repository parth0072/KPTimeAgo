## Usage

First Step  - `import KPTimeAgo ` to your project

```
let timeAgo = (Date().formatDateAsTimeAgo()) 
print(timeAgo)
//Just now

Other Example
//August 25 at 10:30 AM
//July 25
//September 25
//July 25, 2022
//October 25
//July 25, 2023


```


## CocoaPods Install

Add this to your Podfile.

```
pod 'KPTimeAgo'
```
